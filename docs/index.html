Isolation

Its Own World

Each task lives in a fully isolated workspace — its own git branch, its own terminal session, its own spec. Run multiple agents in parallel without conflicts.

Install in 10 seconds

Single binary with embedded web frontend. No runtime dependencies beyond Git and a terminal multiplexer.

$ curl -sSL https://raw.githubusercontent.com/GarrickZ2/grove/master/install.sh | sh
Auto-detects platform (macOS/Linux) and architecture. GUI included on macOS.
$ cargo install grove-rs
Installs TUI + Web UI. Does not include native GUI.
$ cargo install grove-rs --features gui
Includes native macOS GUI via Tauri 2. Run grove gui to launch.
$ git clone https://github.com/GarrickZ2/grove.git
$ cd grove && cargo build --release
$ cp target/release/grove /usr/local/bin/

Then just run:

$ cd your-project && grove # TUI
$ cd your-project && grove web # Web UI
$ cd your-project && grove gui # Desktop GUI
