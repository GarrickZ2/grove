# Grove Icons

This directory should contain the application icons in various formats:

- `32x32.png` - 32x32 pixels PNG
- `128x128.png` - 128x128 pixels PNG
- `128x128@2x.png` - 256x256 pixels PNG (for retina displays)
- `icon.icns` - macOS icon bundle
- `icon.ico` - Windows icon

You can generate these icons from a source image using:
- https://www.npmjs.com/package/@aspect-build/rules_tauri
- https://tauri.app/distribute/icons/

For now, placeholder icons are provided for development.
